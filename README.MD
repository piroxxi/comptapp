Comptapp est un système complet de gestion de comptes. Permettant de gérer une liste précise des dépenses et des recettes, et permettant de maitriser son budget.

Une version de démonstration en version 1.2 est disponible ici Venez vite l'essayer!

Et si vous voyer un bug, ou que vous avec une idée pour améliorer la qualité de cette application, n'hésitez pas à partager ici : http://code.google.com/p/comptapp/issues/list

Bonne comptabilité :)
